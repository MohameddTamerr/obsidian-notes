Conditions

- objective function **linear**
    
- constraints **linear**
    
- variables **continuous**
    

Example

max 3x1 + 5x2  
  
s.t  
x1 + 2x2 ≤ 10  
2x1 + x2 ≤ 12  
x1 , x2 ≥ 0

Applications

- production planning
    
- resource allocation
    
- portfolio management
    
- diet problems
